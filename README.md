# 🌸 Bloom & Note

A digital space for building customized bouquets (just because the world can always use more flowers!)

There’s something special about receiving flowers, so I created Bloom & Note to make that moment happen more often. It’s a creative workspace where you can craft a customized bouquet and send it to the people you care about, no matter the time or distance.

Let's spread the happiness! 🌼

## Developing

Install the dependencies:

```bash
npm install
```

Start the development server with HMR:

```bash
npm run dev
```

Your application will be available at `http://localhost:5173`.

## Building for Production

Create a production build:

```bash
npm run build
```
